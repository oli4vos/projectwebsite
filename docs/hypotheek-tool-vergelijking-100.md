# Hypotheek tool vergelijking - 100 testsets

Publieke calculators geven niet altijd exact dezelfde uitkomst terug aan de voorkant.
Daarom combineert deze vergelijking:
- exacte uitkomst van onze engine;
- publiek zichtbare gedragspatronen per aanbieder;
- een Independer-vuistregel als sanity check;
- interne normtests op inkomen, collateral en koopbudget.

De kolommen ABN AMRO, Rabobank en ING zijn hier observaties van publiek zichtbare flows, geen gevalideerde expected values.
